# 🌍 Travel Explorer – Smart Travel Companion

## ✨ Overview

**Travel Explorer** is a modern web application that helps users explore any destination with essential travel insights — all in one place.

Instead of visiting multiple websites, users can simply enter a city or country and instantly get:

* 🌦 Weather details
* 🌎 Country information
* 💱 Currency exchange rates
* 🗺 Key travel insights

This project focuses on leveraging **JavaScript, Fetch API, and Promises** to combine multiple data sources into a seamless user experience.

---

## 🚀 Features

* 🔍 **Search Any Destination**
  Enter a city or country to explore travel details instantly.

* 🌦 **Live Weather Data**
  Get real-time temperature, conditions, and weather insights.

* 🌍 **Country Information**
  View population, region, capital, and more.

* 💱 **Currency Exchange Rates**
  Understand conversion rates for better travel planning.

* ⚡ **Fast & Dynamic UI**
  Smooth data fetching using async/await and Promise handling.

---

## 🧠 Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & responsive design
* **JavaScript (Vanilla JS)** – Core logic
* **Fetch API** – Data fetching
* **Promises / Async-Await** – Handling multiple API calls

---

## 🔗 APIs Used

* 🌦 Weather API (OpenWeather or similar)
* 🌍 REST Countries API
* 💱 Currency Exchange API

---

## ⚙️ How It Works

1. User enters a destination (city/country)
2. Multiple API calls are triggered using `fetch()`
3. Data is handled using:

   * `async/await`
   * `Promise.all()`
4. Combined results are displayed dynamically on the UI
